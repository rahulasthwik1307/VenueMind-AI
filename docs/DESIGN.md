# Design System

## Purpose

VenueMind AI is a premium stadium operations platform for FIFA World Cup 2026. The interface must feel like enterprise software used by professional operations teams, not a generic AI dashboard or hackathon project.

---

# Design Philosophy

Design should communicate:

- Trust
- Clarity
- Speed
- Intelligence
- Calm
- Precision

Avoid decorative UI. Every element must have a purpose.

# Visual Style

Inspired by

- Apple
- Linear
- Vercel
- Stripe Dashboard
- Notion

Take inspiration only for hierarchy, spacing, and interactions. Never copy layouts.

---

# Typography

Primary Font

- Geist

Fallback

- Inter

Hierarchy

- Large clear headings
- Comfortable body text
- Consistent spacing
- Maximum 3 font weights

---

# Color Palette

Primary

Deep Emerald

Secondary

Steel Blue

Accent

Warm Amber

Background

Soft Off White

Cards

Pure White

Success

Green

Warning

Amber

Error

Red

Color should guide attention, never decorate.

---

# Layout

- Clean dashboard layout
- Large whitespace
- Consistent 8px spacing system
- Maximum content width 1440px
- Responsive for Desktop, Tablet and Mobile
- Never overcrowd screens

---

# Components

Every component should have

- Clear hierarchy
- Soft border radius
- Soft shadows
- Consistent spacing
- Visible hover state
- Visible focus state

Reuse existing components before creating new ones.

---

# Animations

Use Framer Motion only.

Animations should

- explain
- guide
- provide feedback

Avoid decorative animations.

Duration

150ms–300ms

Respect prefers-reduced-motion.

---

# Icons

Use Lucide React only.

Icons support content.

Never replace text with icons.

---

# Accessibility

Always include

- Keyboard navigation
- Focus rings
- Semantic HTML
- ARIA labels
- Minimum touch target 44px
- WCAG AA

---

# Responsive

Every screen must support

Desktop

Tablet

Mobile

Never hide important functionality.

---

# UI Quality Rules

Always generate

- Premium layouts
- Clean typography
- Consistent spacing
- Modern cards
- Enterprise appearance

Never generate

- Generic AI dashboards
- Random gradients
- Neon colors
- Heavy glassmorphism
- Excessive shadows
- Cluttered layouts
- Inconsistent spacing
- Random border radius
- Oversized icons
- Decorative animations

---

# Design Review Checklist

Before completing any UI

✓ Premium appearance

✓ Consistent typography

✓ Consistent spacing

✓ Accessible

✓ Responsive

✓ Smooth animations

✓ Reusable components

✓ Matches existing design system

✓ No visual clutter

✓ Feels like one product

If a new screen does not match the existing design language, redesign it before implementation.
